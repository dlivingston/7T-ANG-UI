# 7T-ANG-UI

7Tunnels ANG User Interface

## Setup 
	npm install

## Run Express & Socket.io Websocket Server

    nodemon server.js

## Run Gulp & Browserify Dev Tasks
	
    gulp
    
__or__
    
    npm start
     

## Build for Deployment
	
    gulp build --env production

__or__

    npm run build
